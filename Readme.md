# Node.js basics

## Run tests

You can test with the npm scripts below:

#### File system (src/fs)

```bash
npm run fs:create
npm run fs:copy
npm run fs:rename
npm run fs:delete
npm run fs:list
npm run fs:read

```

#### Command line interface(src/cli)

```bash
npm run cli:env
npm run cli:args
```

#### Modules(src/modules)

```bash
npm run modules
```

#### Hash (src/hash)

```bash
npm run hash
```

#### Streams (src/streams)

```bash
npm run streams:read
npm run streams:write
npm run streams:transform
```

#### Zlib (src/zip)

```bash
npm run zip:compress
npm run zip:decompress
```

#### Worker Threads (src/wt)

```bash
npm run wt
```

#### Child Processes (src/cp)

```bash
npm run cp -- add your args here
```
